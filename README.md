# Personal-Portfolio-Website

> A minimal, responsive developer portfolio built with **vanilla web technologies**. Currently in active development.

This repository contains the foundational code for my personal website. It is structured around a modern, modular UI design, built completely without heavy frontend frameworks to prioritize speed, flexibility, and easy deployment via static hosting (such as GitHub Pages).

## Architecture & Capabilities

As this project evolves, the core development focuses are:

* **Modular Layout System:** Utilizes CSS Grid to create a clean, highly adaptable "bento" style interface that can seamlessly house various dynamic widgets and content blocks.
* **Vanilla Tech Foundation:** Built entirely with HTML5, CSS3, and modern Vanilla JavaScript (ES6+), ensuring a lightweight footprint and zero-dependency maintenance.
* **Dynamic Integration Ready:** Includes foundational JavaScript logic designed to handle client-side external API fetching, live DOM updates, and interactive UI micro-animations.
* **Theming Engine:** Features a built-in Light/Dark mode toggling system powered by CSS custom properties for smooth, user-controlled visual transitions.

## Tech Stack

* **Structure:** HTML5
* **Styling:** CSS3 (CSS Grid, Flexbox, Custom Properties, Keyframe Animations)
* **Logic:** Vanilla JavaScript
* **Deployment (Planned):** GitHub Pages

## Local Development

Because this project relies entirely on vanilla web technologies, there are no package managers, dependencies to install, or build steps required.

1. Clone the repository:
   ```bash
   git clone [https://github.com/your-username/Personal-Portfolio-Website.git](https://github.com/ChristosGkovaris/Personal-Portfolio-Website.git)
   ```
2. Navigate to the project directory:
   ```bash
   cd Personal-Portfolio-Website
   ```
3. Open `index.html` directly in your browser of choice. 
   * *Note: For the best development experience, using a local server like VS Code's **Live Server** extension is recommended to enable hot-reloading.*
